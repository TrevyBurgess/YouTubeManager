# YouTube Manager

YouTube is an amazing platform for publishing content.

However, publishing requires plenty of management. Each creator is expected to manage their content. 

This tool is designed to automate some of these tasks.

## 1. Upload Page

You can upload content, as well as edit descriptions, etc.

By default, videos are private. Once YouTube scans a video, you get a report.

### 1.1. Automatic publishing

You can set automatic publishing. However, this app will only publish if you says there are no issue.

## 2. Management Page

On this page you can manage your existing videos

## 3. Moderating Comments

An essential for all creators is managing comments.

Things that hurt your channel:

1. Negative comments
1. Advertising

Given a video, AI will monitor all comments and delete all comments that violate the user's Terms of Service.

### 3.1. Terms of Service

This is where the user defines what comments are acceptable and what are not acceptable.

The app comes with default templates, with the opportunity of sharing templates with the community.

## Settings

Settings will be stored on a folder on the User's Google Drive.
